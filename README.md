A **Full Stack Web Application** that helps devotees easily discover temples, explore temple information, and search temples based on **name, location, or deity**.

This project simplifies temple discovery by providing a clean and structured platform where users can quickly find temples and view important information.

---

# 📌 Project Overview

Darshan Ease allows users to search temples using keywords like **temple name, deity, or location**.

The system uses a **React frontend**, **Node.js backend**, and **MongoDB database** to efficiently fetch and display temple data.

The backend processes search queries and returns matching temple results to the frontend in real time.

# ⚙ Installation

### Clone Repository

```
git clone https://github.com/Shivam22601/DARSHAN-EASE-.git
```

### Install Backend

```
cd backend
npm install
npm start
```

### Install Frontend

```
cd frontend
npm install
npm start
```
